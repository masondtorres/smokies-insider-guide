# Smoky Insider Visual Photo Sources v1

Verified: 2026-07-21

The visual prototype uses National Park Service photographs as decorative page imagery. NPS image detail pages state that images credited to NPS without a copyright symbol are public domain. Images credited to another entity must not be presumed public domain.

## Rights rule

Source: https://www.nps.gov/media/photo/gallery-item.htm?gid=88774C33-1DD8-B71C-0717CFEB07AF4BDB&id=28de6b98-1dd8-b71c-073d-1ffaf7d178b7
